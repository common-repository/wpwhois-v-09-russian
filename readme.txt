Provides usage of Whois service in WP (fully Russian)

== Description ==

Although it works as widget, it can be used from template. It is a good plugin for russian-speaking users of 

WordPress.



== Installation ==

1. Upload `wpwhois.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<? my_domain_search() ?> in your templates. Or place WPHOIS widget on sidebar. Don`t use both methods at 

the same time.
